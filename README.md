# NovusVital MVP Mock

Premium static MVP mockup for partner and investor demos.

## Live Demo

After GitHub Pages is enabled and deployment finishes, your site will be available at:

https://beatviral.github.io/NovusVital/

## What is included

- Editorial luxury brand direction (clinical premium look)
- Interactive intake modal preview
- Curated marketplace section with dynamic cards
- Outcomes dashboard section
- Partner console snapshot
- Mobile-responsive layout
- Scroll reveal animations

## Deployment

This repo includes a GitHub Actions workflow at `.github/workflows/pages.yml` that automatically deploys to GitHub Pages when code is pushed to `main`.

## Local preview

Open `index.html` directly in your browser.
